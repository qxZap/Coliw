CoLiW este o simpla aplicatie web conceputa sa preia siruri de caractere
introduse in interfata (front-end) capabila sa paseze mai departe sirul 
introdus de client catre partea de back-end care va parsa (printr-un parser
custom), analiza si procesa respectivul sir de caractere sub forma unei 
sintaxe de comanda ce suporta comenzi mutiple. Aplicatia are ca scop 
interpretarea corecta si sa suporta inlantuiri de comenzi si pipe-uri ce 
vor duce la efectul dorit in urma executiei lor, lasand afisat in istoricul
interfetei un output relevant.

Aplicatia va folosi tehnologiile Python, HTML5, CSS, js/jQuery, Apache si MySql.
Coliw va fi impartita in doua mari module:

    1. Interfata (front-end) ce va fi o consola statica, cuprinzand tot
       ecranul, ce ca trimite comenzi catre un API (back-end) atunci cand 
       este apasata tasta <Enter>. In urma raspunsului se va afisa  pe ecran
       un text de iesire relevant primit de la server, ca raspuns la comanda
       introdusa.
       Interfata va suporta interactiuni similare oricarei console 
       (ex.: arrow-up/down pentru rasfoirea in comenzile precedente si
       scroll: pentru a naviga prin istoricul de comenzi).

    2. API (back-end) ce va fi responsabil pentru primirea comenzii si de
       asemenea, executia ei. Va garanta obtinerea efectului dorit in urma
       executiei respectivelor comenzi, indiferent de complexitate si va avea
       grija sa furnizeze un raspuns relevant interfetei indiferent daca
       executia a avut loc cu succes sau nu.

        a. Un set de Views ce vor interpreta rutele inregistrare si vor delega
           comanda unei biblioteci special desemnata pentru procesarea ei.
           Va intoarce un template html populat de rezultat.
        b. Un set de Models pentru stocarea datelor: User
        c. Un set de Forms pentru afisarea de campuri in interfata si salvarea
           intrarilor utilizatorilor in modele.
